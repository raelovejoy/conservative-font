# Conservative

**Conservative** is a parody typeface in which every character renders identically.

Regardless of what you type—letters, numbers, punctuation—the output is the same.
The font preserves uniformity, consistency, and tradition across all inputs.

## Details
- Format: TrueType (`.ttf`)
- Style: Regular
- Glyph count: 257
- Color glyphs: No
- Footprint: ~52 KB

## Usage
Install the font and select **Conservative** in any application.
Type normally.
